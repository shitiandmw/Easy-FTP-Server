### Development Guide

#### Environment Setup

```bash
# Make sure you have installed
- Go 1.18+
- Node.js 14+
- npm or yarn
- Wails CLI
```

#### Development Mode

```bash
# Install frontend dependencies
cd frontend
npm install

# Start development server
cd ..
wails dev
```

#### Build

```bash
# Generate release version
wails build
```
